# Airflow GenAI Test Experiments

This repository documents my hands-on experimentation with Apache Airflow, focusing on DAG development, debugging workflows, and exploring GenAI-related operators as part of my preparation for Google Summer of Code (GSoC) 2026.

---

## 📌 Overview

The goal of this project is to build a strong practical understanding of Airflow's architecture and workflows by:

- Creating and executing Directed Acyclic Graphs (DAGs)
- Debugging import and dependency issues in Airflow environments
- Testing task execution and operator behavior
- Exploring GenAI operator integrations within Airflow
- Understanding provider compatibility and version constraints

---

## ⚙️ Tech Stack

- Python 3.11
- Apache Airflow (Local Setup - WSL)
- Git & GitHub
- Linux (WSL environment)
